# Flask Todo App (JWT Auth)

A simple Flask-based Todo application with user authentication using JWT stored in cookies, SQLAlchemy for persistence, and basic CRUD functionality for todos.

This project is intended for learning and small-scale use, demonstrating:

User registration & login

JWT authentication with cookies

User-specific todo management

RESTful update/delete endpoints

## Features

User registration and login

JWT authentication using Flask-JWT-Extended

JWT stored securely in HTTP cookies

User-specific todo lists

Create, update, and delete todos

SQLite database using SQLAlchemy ORM

Server-side rendered templates (Jinja2)

## Tech Stack

Python 

Flask

Flask-SQLAlchemy

Flask-JWT-Extended

SQLite

Jinja2 templates
